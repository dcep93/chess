mobile:
dragging on the board doesn't yet work for mobile
use "best move" to test yourself in your head
keyboard shortcuts:
left: undo
right: redo
up: best move
down: new game (switch sides)
enter: different_move
shift+move: create novelty
space: clear novelty
log fields:
(mouse over the log for a list of common moves)
(click the log to jump)
s/move score (max 420)
p/move frequency / 100
ww/odds that white wins
d/odds that the game is a draw
t/total number of games
novelty:
if you have a favorite move in a position
that isn't necessarily the best,
create a novelty to define a "best move"
in the future, using "best move" will play this
instead of the default "best move"
hold shift when moving to create a novelty
